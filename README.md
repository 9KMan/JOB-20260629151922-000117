# BPA Pipeline

Business Process Automation — Web Scraping + Data Pipeline (MVP).

## Stack

- Python 3.12
- FastAPI + Uvicorn
- PostgreSQL 15 (async SQLAlchemy + asyncpg)
- Playwright
- APScheduler
- python-telegram-bot
- gspread (Google Sheets)
- Alembic (migrations)
- structlog (logging)
