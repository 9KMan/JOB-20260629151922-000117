# BPA Pipeline

Business Process Automation — Web Scraping + Data Pipeline (MVP).

## Stack

- Python 3.12 + FastAPI + Uvicorn
- Playwright (Chromium) + httpx
- PostgreSQL 15 + SQLAlchemy (async) + Alembic
- APScheduler (in-process cron)
- python-telegram-bot (optional notifications)
- gspread + google-auth (optional Sheets export)
- structlog + tenacity
